# Professional Portfolio Website Prompt (HTML & CSS Only)

Here's a comprehensive prompt for creating a professional portfolio website using only HTML and CSS:

## HTML & CSS Portfolio Website Requirements

**Objective:** Create a clean, responsive portfolio website to showcase professional work, skills, and contact information.

### HTML Structure Requirements:
1. Semantic HTML5 elements throughout
2. Responsive meta tags in head section
3. Six main sections:
   - Header with navigation
   - Hero/Introduction section
   - About Me section
   - Skills/Technologies section
   - Projects/Portfolio section
   - Contact section
   - Footer

### CSS Styling Requirements:
1. Mobile-first responsive design
2. Clean, modern aesthetic with appropriate whitespace
3. CSS custom properties for colors and fonts
4. Smooth hover effects and transitions
5. Flexbox and/or Grid for layout
6. Media queries for responsive breakpoints

### Section Details:

**1. Header/Navigation:**
- Logo/name on left
- Navigation links (Home, About, Skills, Projects, Contact)
- Sticky positioning
- Subtle hover effects on nav items

**2. Hero Section:**
- Professional headshot or placeholder
- Name and professional title
- Brief tagline (1-2 sentences)
- Call-to-action button (e.g., "View My Work")

**3. About Section:**
- Professional bio paragraph
- Education/certifications
- Career highlights
- Personal interests (optional)

**4. Skills Section:**
- Categorized skills (e.g., Frontend, Backend, Design)
- Visual representation (progress bars, icons, etc.)
- Technology logos/icons where appropriate

**5. Projects Section:**
- 3-6 project cards with:
  - Project image/thumbnail
  - Project title
  - Brief description
  - Technologies used
  - Links to live demo/code (as buttons/icons)

**6. Contact Section:**
- Contact form with:
  - Name field
  - Email field
  - Message textarea
  - Submit button
- Alternative contact methods (email, social media)

**7. Footer:**
- Copyright information
- Social media links/icons
- Back-to-top button

### Technical Requirements:
- No JavaScript - pure HTML/CSS only
- Valid HTML5 and CSS3
- Accessible (proper alt text, ARIA where needed)
- Cross-browser compatible
- Optimized for performance

### Visual Style Guidelines:
- Professional color scheme (max 3-4 colors)
- Readable typography (Google Fonts recommended)
- Consistent spacing and alignment
- Subtle animations/transitions for interactivity
